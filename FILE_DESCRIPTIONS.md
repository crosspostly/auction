# Описание файлов проекта VK Аукционного Бота

## Основные файлы

### Code.gs
Основной файл приложения, содержащий:
- Обработчики веб-хуков (`doPost`, `doGet`)
- Маршрутизацию событий (`routeEvent`)
- Обработчики основных событий ВКонтакте (`handleWallPostNew`, `handleWallReplyNew`, `handleMessageNew`)
- Функции управления аукционом (`finalizeAuction`, `processNotificationQueue`)
- Функции тестирования и UI (`onOpen`, меню приложения)

### Vk.gs
Файл для взаимодействия с VK API:
- Функция `callVk` с обработкой ошибок и экспоненциальным ретраем
- Функции отправки сообщений и комментариев
- Функции получения информации о пользователях
- Настройка Callback API сервера

### Sheets.gs
Файл для работы с Google Sheets:
- Конфигурация структуры таблиц
- Функции чтения/записи данных в листы
- Функции логирования
- Функции управления лотами, ставками и победителями

### Monitoring.gs
Файл для независимого мониторинга:
- Модуль `Monitoring` для записи ключевых событий
- Функции логирования в лист "Статистика"

## Дополнительные функции

### AuctionSimulator.gs
Файл симулятора аукциона:
- Генерация тестовых постов-лотов
- Имитация ставок от разных пользователей
- Сценарии тестирования различных ситуаций
- Управление ежечасным запуском симуляции

### Tests.gs
Файл модульных тестов:
- Тесты отдельных компонентов системы
- Проверка функций парсинга
- Проверка валидации ставок
- Тесты основных сценариев

### AdditionalTests.gs
Файл интеграционных тестов:
- Тесты полного цикла аукциона
- Комплексные сценарии тестирования
- Проверка взаимодействия компонентов

### SystemMonitor.gs
Файл системы мониторинга:
- Проверка здоровья системы
- Автоматическое восстановление
- Обнаружение аномалий
- Функции диагностики

### PeriodicTasks.gs
Файл периодических задач:
- Функции, запускаемые по расписанию
- Очистка старых данных
- Периодические проверки

### NewFunctions.gs
Файл новых функций:
- Вспомогательные функции
- Дополнительная логика

### SubscriptionChecker.gs
Файл проверки подписки:
- Функции проверки членства в группе
- Валидация участников аукциона

### WallReplyHandlers.gs
Файл обработчиков комментариев:
- Обработка редактирования комментариев (`handleWallReplyEdit`)
- Обработка удаления комментариев (`handleWallReplyDelete`)
- Логика обновления/удаления ставок

### TestComponents.gs
Файл тестирования компонентов:
- Функция проверки наличия всех компонентов
- Диагностика системы

## HTML файлы

### SimpleInstructions.html
HTML-страница с простой инструкцией для администраторов:
- Понятные пошаговые инструкции
- Описание всех настроек
- Руководство по устранению неполадок

### AuthSettings.html
HTML-форма для настройки авторизации:
- Ввод токенов и реквизитов
- Защита паролем

### Login.html
HTML-страница входа:
- Аутентификация администратора
- Защита доступа к настройкам
