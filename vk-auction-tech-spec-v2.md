# ТЕХНИЧЕСКАЯ СПЕЦИФИКАЦИЯ (v3.1)

## 1. ОБЩИЕ СВЕДЕНИЯ
Система реального времени для автоматизации аукционов в группах ВКонтакте.

## 2. ОСНОВНЫЕ ТРЕБОВАНИЯ
- **Реакция**: < 2 секунд.
- **Уникальность**: По `post_id` (разные посты = разные лоты).
- **Время**: Строго Московское (GMT+3).
- **Безопасность**: Только токен группы для операционной деятельности.

## 3. ЛОГИКА СТАВОК
| Сценарий | Действие бота |
|----------|---------------|
| Успешная ставка | Запись в базу, молчание в ВК (лидер виден в таблице) |
| Ставка перебита | Ответ в комментарии: "Ваша ставка перебита" |
| Ошибка (сумма/шаг) | Ответ в комментарии с указанием правильной суммы |
| Дубликат события | Игнорирование (защита CacheService + DB) |

## 4. ФИНАЛИЗАЦИЯ
- Автоматический запуск мониторинга при публикации поста.
- Авто-закрытие по дедлайну.
- Отчет администраторам в ЛС.
- Рассылка реквизитов победителям в ЛС (опционально).

## 5. ПОДДЕРЖКА И ОТЛАДКА
- Лист **Incoming**: 100 последних запросов.
- Лист **Logs**: Полная история событий, свежие записи сверху.
- Защита от смещения колонок в Sheets.
